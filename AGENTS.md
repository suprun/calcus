# AGENTS.md

Інструкції для ШІ агента, що працює з цим репозиторієм.

## Загальні правила

- Перед змінами уважно перевіряй релевантні файли проєкту.
- Не змінюй сторонні файли без прямої необхідності.
- Для пошуку по коду використовуй швидкі інструменти на кшталт `rg`, якщо вони доступні.
- Тримай зміни мінімальними й прив'язаними до конкретного запиту.
- Відповідай українською мовою.
- Пиши код у стилі, який уже використовується в проєкті.
- Не змінюй архітектуру без явної потреби.
- Не видаляй існуючий функціонал без пояснення.
- Перед великими змінами коротко пояснюй план.

## Стиль коду

- JavaScript: дотримуйся Airbnb Style Guide.
- Назви змінних мають бути зрозумілими.
- Не додавай зайві абстракції.

## Коментарі

- Коментуй тільки неочевидну логіку.
- Не пиши коментарі типу `# increment counter`, якщо дія очевидна з коду.

## Безпека

- Не виводь секретні дані в лог.

## Правила роботи з промптами

- Plan mode: / планування: - режим планування без редагування коду
- Ask mode: / запитання: - режим відповідей на запитання асистентом без редагування коду

## Git

- Не змінюй файли, які не стосуються задачі.
- Не форматуй увесь проєкт без потреби.
- Уникай великих не пов’язаних між собою змін.

## Дані, які не треба редагувати, а лише використовувати за потребою

- calc.html не редагувати, а використовувати як референс, якщо на нього вказується в задачі для отримання ідей чи рішень.

# Перевірка результатів
- не потрібно запускати агента для перевірки результатів в браузері, а лише перевіряти результати вручну, якщо на це вказується в задачі.
